# Emerge

Emerge is a minimal AI shell project focused on:

- `read` / `write` / `bash` execution primitives
- A-track pipeline evolution
- state-delta based context denoising and token reduction

## Repository layout

- `docs/superpowers/specs/` - product and architecture specifications
- `docs/superpowers/plans/` - implementation plans
- `references/` - external reference codebases (as git submodules)

## Reference sources

This repository keeps Claude Code source only as reference material under `references/` via git submodules, so we can add more reference projects later without mixing them into Emerge core code.
